## JDK

下载链接：https://www.oracle.com/cn/java/technologies/downloads/#java17

命令

```shell
1.安装jdk
tar -zxvf jdk-*.tar.gz
2.重命名
 mv jdk-17.0.10 jdk17
3.移动
mv jdk17 /usr/local/
4.修改配置文件
cd /usr/local/
vim /etc/profile
(按a或者i 进去编译模式  :wq保存退出)
5.添加配置
export JAVA_HOME=/usr/local/jdk17
export CLASSPATH=$:CLASSPATH:$JAVA_HOME/lib/
export PATH=$PATH:$JAVA_HOME/bin
cat /etc/profile（查看是否更改成功）
6.刷新配置
source /etc/profile
7.检查是否安装成功
java -version
```

## docker

菜鸟教程：https://www.runoob.com/docker/centos-docker-install.html

#### 1.手动安装

1.卸载旧版本

```sh
sudo yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-engine
```

2.安装所需软件包

```sh
sudo yum install -y yum-utils \
  device-mapper-persistent-data \
  lvm2
```

3.设置仓库 阿里云

```sh
sudo yum-config-manager \
    --add-repo \
    https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
```

4.安装 Docker Engine-Community

```sh
sudo yum install docker-ce docker-ce-cli containerd.io docker-compose-plugin
```

#### 2.启动docker

```sh
sudo systemctl start docker
docker version 验证成功
```

用户设置Docker需要用户具有sudo权限，为了避免每次命令都输入sudo，可以把用户加入到Dokcer用户组

```sh
sudo usermod -aG docker $USER
```

查看镜像文件

```sh
docker images
docker ps 查看所有进程
```

设置开机自启

```sh
systemctl enable docker
```

镜像加速

```shell
sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
    "registry-mirrors": ["https://3n4m4jry.mirror.aliyuncs.com"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker
```

#### 卸载docker

```sh
删除安装包：
yum remove docker-ce
删除镜像、容器、配置文件等内容：
rm -rf /var/lib/docker
```

### docker 安装mysql

安装mysql

```shell
docker pull mysql:8.0
```

启动mysql

```shell
docker run -p 3306:3306 --name mysql \
-v /mydata/mysql/log:/var/log/mysql \
-v /mydata/mysql/data:/var/lib/mysql \
-v /mydata/mysql/conf:/etc/mysql \
-e MYSQL_ROOT_PASSWORD=root \
-d mysql:8.0

docker run -p 3309:3306 --name mysql \
-e MYSQL_ROOT_PASSWORD=root \
-d mysql:8.0

docker run -p 3309:3306 --name mysql3309 \
-v /mydata/mysql3309/log:/var/log/mysql \
-v /mydata/mysql3309/data:/var/lib/mysql \
-v /mydata/mysql3309/conf:/etc/mysql \
-e MYSQL_ROOT_PASSWORD=root \
-d mysql:8.0

//用这个就可以
docker run -p 3306:3306 --name mysql -e MYSQL_ROOT_PASSWORD=cyx -d mysql:8.0

docker start mysql
```

项目上云

1.安装 Node.js 16 和 npm

```sh
安装 Node.js 的安装包
curl -fsSL https://rpm.nodesource.com/setup_16.x | sudo bash -
 安装Node.js 和 npm
sudo yum install -y nodejs
确认安装成功
node -v
npm -v
```

2.在node环境下安装 `http-server`

```sh
npm install -g http-server
```

3.启动http服务器

```sh
http-server -p 5173 
将 http-server 在后台运行
nohup http-server -p 5173 > /dev/null 2>&1 &
```
