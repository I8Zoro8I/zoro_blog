## 📖 如何在洛雪音乐中导入音源？

1. **复制链接**：在上方列表中选择一个音源（例如 SixYin 或 LX），复制对应的 `.js` 链接。
2. **打开设置**：打开你的洛雪音乐桌面端或移动端，进入 **设置**。
3. **找到自定义源**：在设置列表中找到 **基本设置** -> **自定义源管理**（或音乐来源）。
4. **在线导入**：点击 **【在线导入】**，将刚刚复制的链接粘贴进去，点击确定。
5. **勾选启用**：导入成功后，勾选刚刚添加的音源，看到状态显示为“初始化成功”即可开始畅听！
