# mac 软件推荐（Windows 用户转 Mac 必备小工具）

我是一个长期使用 Windows 的用户。

真正开始接触 Mac，其实是上班之后。

刚切过来的那段时间，说实话并不适应。

甚至会有一种：

> “这系统怎么什么都不顺手？”

的感觉。

比如：

- 没有右键新建
- 鼠标滚轮方向不习惯
- Finder 没资源管理器好用
- 软件关闭逻辑奇怪
- 多窗口切换不舒服
- 软件卸载不干净
- 外接鼠标体验一般

但后来慢慢发现：

Mac 的问题很多不是系统本身。

而是：

> 缺少适合自己的工具链。

装对软件之后，整个体验会舒服非常多。

所以这篇文章。

主要整理一些：

# 特别适合 Windows 用户转 Mac 的软件

尤其适合：

- 程序员
- 办公党
- 学生
- 重度电脑用户

---

# 1. RunCat

一个非常有意思的小工具。

它会在状态栏显示一只奔跑的小猫（或者别的动物）。

CPU 占用越高。

它跑得越快。

---

## 它有什么用？

严格来说：

> 没什么生产力价值。

但它属于：

> “会让你心情很好”的软件。

尤其是程序员。

当你：

- 开 Docker
- 跑前端项目
- IDEA 索引
- 编译项目
- Chrome 开几十个标签页

的时候。

你会非常直观看到：

> “哦，CPU 炸了。”

---

## 适合人群

- 程序员
- 喜欢桌面小组件的人
- 想实时观察 CPU 状态的人

---

# 2. 右键助手（RightMenu Master）

Windows 用户刚切到 Mac 时。

最难受的一件事：

# Finder 右键功能太少。

很多 Windows 上习惯的操作。

Mac 原生都没有。

比如：

- 新建 txt 文件
- 剪切
- 复制路径
- 终端打开
- 压缩解压
- 快速创建文件

而 `RightMenu Master` 基本能补齐这些体验。

---

## 为什么推荐？

因为它会让 Finder 更像：

> “正常的 Windows 文件管理器”。

属于：

# Windows 用户幸福感提升特别明显的软件。

---

# 3. BetterMouse

Mac 原生对第三方鼠标支持真的一般。

尤其是：

- 罗技
- 雷蛇
- 英菲克
- ATK
- 国产电竞鼠标

很多时候都会出现：

- 鼠标加速度奇怪
- 滚轮方向别扭
- 滚动不顺滑
- 回报率异常
- 自定义按键失效

的问题。

而 `BetterMouse` 基本能解决大部分问题。

---

## 我最喜欢的功能

### 1. 鼠标独立配置

不同鼠标可以：

- 单独 DPI
- 单独滚动速度
- 单独按键
- 单独滚轮方向

这个真的很舒服。

---

### 2. 平滑滚动

很多鼠标在 Mac 上滚动时。

会有一种：

> “一格一格卡着滚”

的感觉。

BetterMouse 调完之后会顺滑很多。

体验提升非常明显。

---

# 4. DockDoor

这是我觉得：

# Windows 用户转 Mac 必装的软件之一。

它主要是：

> 给 Mac 补全类似 Windows 的窗口预览体验。

---

## 为什么会需要它？

如果你长期用 Windows。

应该很熟悉：

- 鼠标移动到任务栏图标
- 自动显示窗口预览
- 快速切换窗口

但 Mac 原生其实没有这种体验。

Dock 只能看到：

> “应用”。

而不是：

> “窗口”。

这对于刚从 Windows 转过来的人来说。

其实很不习惯。

---

## 它最舒服的功能

### 1. Dock 窗口预览

鼠标移动到 Dock 图标时。

会显示当前窗口缩略图。

特别适合：

- 多个 Finder
- 多个 Chrome
- 多个 IDEA 项目
- 多终端窗口

的人。

---

### 2. 更好的窗口切换逻辑

Mac 原生：

```text
Command + Tab
```

很多时候切的是：

> 应用

不是：

> 窗口。

而 DockDoor 会更接近 Windows 的逻辑。

多窗口切换会舒服很多。

---

## 我的实际感受

刚开始用 Mac 时。

我最不适应的就是：

# 多窗口管理。

后来装了 DockDoor 之后。

整体会更像：

> “Windows + Mac 的混合体验”。

舒服非常多。

---

# 5. Slidepad

这是我目前最喜欢的 Mac 软件之一。

简单来说：

> “侧边栏版浏览器”。

---

## 它能干什么？

你可以把网页收进侧边栏。

比如：

- ChatGPT
- 飞书
- 微信读书
- GitHub
- 翻译网站
- Notion

然后随时从侧边滑出来。

---

## 为什么特别好用？

对于开发来说。

经常会：

- 一边写代码
- 一边查文档
- 一边看 AI
- 一边查 GitHub

如果全放浏览器。

标签页很容易爆炸。

而 Slidepad 会把这些：

> “高频工具网页”

统一收纳起来。

---

## 特别适合

- 小屏幕 MacBook
- 多任务办公
- 写代码查资料
- AI 重度用户

---

# 6. AppCleaner & Uninstaller

Mac 卸载软件其实并不干净。

很多软件即使拖进垃圾桶。

它的：

- 配置文件
- 缓存
- 日志
- 启动项

依然还在。

久了之后系统会越来越乱。

---

## AppCleaner 的作用

它会自动扫描：

- 软件残留
- 配置文件
- 缓存文件

然后一起删除。

---

## 为什么推荐？

因为很多 Windows 用户以为：

> Mac 拖进垃圾桶 = 卸载。

实际上并不是。

所以这个工具我觉得属于：

# Mac 必备软件。

---

# 7. WailBrew

如果你刚从 Windows 转过来。

那你大概率会经历：

# “Mac 到底怎么装开发环境？”

这个阶段。

而在 Mac 的开发生态里。

# Homebrew 基本是核心。

---

## Homebrew 是什么？

可以理解成：

> Mac 的包管理器。

类似：

- apt
- yum
- scoop
- winget

---

## 为什么推荐 WailBrew？

因为很多新手刚接触：

```bash
brew install xxx
```

会有点懵。

而 `WailBrew` 提供了图形化界面。

你可以：

- 搜索软件
- 安装 brew 包
- 更新软件
- 管理环境

对于新手特别友好。

---

# 8. Scherlock

这是一个我非常推荐的：

# Mac 文件检索工具。

尤其适合：

- 文件很多的人
- 经常找资料的人
- 程序员
- Windows 转 Mac 用户

---

## 为什么需要它？

Mac 原生 Spotlight 虽然很好用。

但有时候：

- 搜索速度一般
- 文件筛选不够直观
- 搜代码不方便
- 中文搜索体验一般
- 大量文件时不够精准

而 `Scherlock` 会更像：

> “增强版 Everything”。

---

## Windows 用户应该很熟悉

如果你以前是 Windows 用户。

那大概率知道：

# Everything

那个几乎秒搜文件的软件。

而 Scherlock 在 Mac 上。

会有一种类似的感觉。

---

## 我最喜欢的地方

### 1. 搜索速度非常快

输入关键字。

基本瞬间就能出结果。

特别适合：

- 下载目录
- 项目文件
- 图片素材
- 文档资料

很多的人。

---

### 2. 更细的文件筛选

支持：

- 文件类型
- 文件时间
- 文件大小
- 文件路径

查东西效率会高很多。

---

### 3. 对程序员特别友好

经常会遇到：

- 忘记项目在哪
- 忘记配置文件名字
- 找不到 sql 文件
- 找不到下载内容

这种情况。

有了 Scherlock。

直接搜关键字就行。

---

# Mac 和 Windows 的差异感受

刚开始接触 Mac 的时候。

我其实非常不适应。

尤其是：

- Finder
- 多窗口逻辑
- 快捷键
- 鼠标操作
- 软件关闭逻辑

都会让 Windows 用户很难受。

---

## 但用久之后会发现

Mac 更像：

> “一个偏向工作流和专注的系统”。

尤其是：

- 多桌面
- 触控板
- 动画一致性
- Unix 生态
- 开发环境

这些地方。

确实会越用越舒服。

---

## 当然

Windows 依然是：

# 游戏和兼容性最强的平台。

所以我现在基本属于：

- 游戏娱乐用 Windows
- 开发办公用 Mac

的状态。

---

# 最后推荐几个我觉得 Mac 必学的东西

## 1. Homebrew

Mac 的核心包管理器。

很多开发环境：

- node
- git
- python
- docker
- mysql

都会用它安装。

属于：

# 学 Mac 基本绕不过去的东西。

---

## 2. Raycast

Mac 神器。

可以理解成：

> 超级增强版 Spotlight。

---

## 它能干什么？

- 搜文件
- 启动软件
- AI
- 剪贴板历史
- 快捷命令
- 窗口管理
- 脚本自动化

很多人用了之后。

甚至开始：

> “尽量不用鼠标”。

---

## 3. Rectangle

免费的窗口管理工具。

让 Mac 拥有类似 Windows 的：

- 左右分屏
- 快捷贴边
- 最大化

体验。

属于：

# Windows 用户过渡神器。

---

# 总结

如果你也是：

# 从 Windows 转到 Mac 的用户。

那我非常建议优先解决：

- 鼠标体验
- Finder 操作
- 窗口管理
- 软件卸载
- 开发环境

这些问题。

因为只要这些舒服了。

Mac 的体验会直接提升一个档次。

很多人不是不适应 Mac。

而是：

> 没装对工具。
