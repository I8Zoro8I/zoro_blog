# Spark SQL、DataFrames 和 Datasets 指南

Spark SQL 是 Spark 用于结构化数据处理的模块。与基础的 Spark RDD API 不同，Spark SQL 提供的接口为 Spark 提供了更多关于数据结构和正在执行的计算的信息。在内部，Spark SQL 利用这些额外信息执行优化。有多种方式可以与 Spark SQL 进行交互，包括 SQL 和 Dataset API。在计算结果时，无论您使用哪种 API/语言来表达计算，都会使用相同的执行引擎。这种统一性意味着开发人员可以根据最自然的方式来表达特定的转换，轻松地在不同 API 之间切换。

本页中的所有示例均使用了 Spark 发行版中包含的示例数据，可以在 `spark-shell`、`pyspark` shell 或 `sparkR` shell 中运行。

## SQL

Spark SQL 的一种用途是执行 SQL 查询。Spark SQL 也可用于从现有的 Hive 安装中读取数据。有关如何配置此功能的更多信息，请参阅[ Hive 表](https://spark.apache.ac.cn/docs/latest/sql-data-sources-hive-tables.html)部分。当在其他编程语言中运行 SQL 时，结果将作为 [Dataset/DataFrame](https://spark.apache.ac.cn/docs/latest/sql-programming-guide.html#datasets-and-dataframes) 返回。您还可以通过[命令行](https://spark.apache.ac.cn/docs/latest/sql-distributed-sql-engine.html#running-the-spark-sql-cli)或 [JDBC/ODBC](https://spark.apache.ac.cn/docs/latest/sql-distributed-sql-engine.html#running-the-thrift-jdbcodbc-server) 与 SQL 接口进行交互。

## Datasets 和 DataFrames

Dataset 是数据的分布式集合。Dataset 是 Spark 1.6 中引入的新接口，它结合了 RDD 的优点（强类型、使用强大的 lambda 函数的能力）与 Spark SQL 优化执行引擎的优势。Dataset 可以从 JVM 对象[构建](https://spark.apache.ac.cn/docs/latest/sql-getting-started.html#creating-datasets)，然后使用函数式转换（`map`、`flatMap`、`filter` 等）进行操作。Dataset API 可用于 [Scala](https://spark.apache.ac.cn/docs/latest/api/scala/org/apache/spark/sql/Dataset.html) 和 [Java](https://spark.apache.ac.cn/docs/latest/api/java/index.html?org/apache/spark/sql/Dataset.html)。Python 不支持 Dataset API，但由于 Python 的动态特性，Dataset API 的许多优势已经具备（例如，您可以自然地通过 `row.columnName` 访问行的字段）。R 语言的情况也类似。

DataFrame 是组织成命名列的 *Dataset*。在概念上，它等同于关系数据库中的表或 R/Python 中的数据框，但在底层具有更丰富的优化。DataFrame 可以从多种[数据源](https://spark.apache.ac.cn/docs/latest/sql-data-sources.html)构建，例如：结构化数据文件、Hive 中的表、外部数据库或现有的 RDD。DataFrame API 可用于 [Python](https://spark.apache.ac.cn/docs/latest/api/python/reference/pyspark.sql/api/pyspark.sql.DataFrame.html#pyspark.sql.DataFrame)、Scala、Java 和 [R](https://spark.apache.ac.cn/docs/latest/api/R/index.html)。在 Scala 和 Java 中，DataFrame 表示为 `Row` 的 Dataset。在 [Scala API](https://spark.apache.ac.cn/docs/latest/api/scala/org/apache/spark/sql/Dataset.html) 中，`DataFrame` 仅仅是 `Dataset[Row]` 的类型别名。而在 [Java API](https://spark.apache.ac.cn/docs/latest/api/java/index.html?org/apache/spark/sql/Dataset.html) 中，用户需要使用 `Dataset<Row>` 来表示 `DataFrame`。

在本文档中，我们将经常把 `Row` 的 Scala/Java Dataset 称为 DataFrame。
