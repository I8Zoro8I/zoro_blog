# 最新方案 项目自动切换node

 创建 .nvmrc自动识别node版本进行切换

2025年12月29日

在项目目录创建 .nvmrc

```bash
cd /your/project
node -v
# 比如 v18.20.2

echo "18.20.2" > .nvmrc   #只写一个8
# 或
echo "lts/*" > .nvmrc  #当前最新的LTS[长期支持]版本的node
```

.nvmrc 内容示例：

```text
18.20.2
```

### 配置「自动读取 .nvmrc」

#### 👉 zsh（macOS 默认）

编辑 ~/.zshrc，加上：

```bash
autoload -U add-zsh-hook

load-nvmrc() {
  local node_version="$(nvm version)"
  local nvmrc_path="$(nvm_find_nvmrc)"

  if [ -n "$nvmrc_path" ]; then
    local nvmrc_node_version="$(nvm version "$(cat "$nvmrc_path")")"

    if [ "$nvmrc_node_version" = "N/A" ]; then
      nvm install
    elif [ "$nvmrc_node_version" != "$node_version" ]; then
      nvm use
    fi
  fi
}

add-zsh-hook chpwd load-nvmrc
load-nvmrc
```

📌 效果：

- cd 项目目录 → 自动 nvm use
- 离开目录 → 不会反复执行
- 版本不存在 → 自动 nvm install

------

#### 👉 bash（Linux / 旧 macOS）

编辑 ~/.bashrc：

```bash
cd() {
  builtin cd "$@" || return
  if [ -f .nvmrc ]; then
    nvm use --silent
  fi
}
```
