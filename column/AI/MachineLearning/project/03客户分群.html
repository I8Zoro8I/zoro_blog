机器学习 - 客户分群 ​

在当今数据驱动的商业世界中，理解客户是成功的关键。

然而，当你的客户数量达到成千上万甚至百万级别时，手动分析每个客户的特征和行为模式变得不切实际。

这时，机器学习技术，特别是 无监督学习 中的聚类算法，就成为了一个强大的工具。

客户分群，也称为客户细分，其核心目标是将庞大的客户群体划分为若干个具有相似特征的子群体。这就像一位经验丰富的店主，不再将顾客视为一个模糊的整体，而是能清晰地识别出追求性价比的家庭主妇、热衷新品的技术发烧友和注重服务体验的高端客户等不同群体。通过对不同群体采取针对性的营销、服务和产品策略，企业可以显著提升运营效率和客户满意度。

本文将带你一步步完成一个完整的客户分群实战项目。我们将使用经典的 K-Means 聚类算法，对一个模拟的零售客户数据集进行分析，从数据理解到模型评估，最终获得具有商业洞察力的分群结果。

理解聚类分析与 K-Means 算法 ​

在开始实战之前，我们需要理解即将使用的核心工具。

什么是聚类分析？ ​

聚类分析是一种无监督学习方法。与监督学习（如预测房价或识别猫狗图片）不同，聚类算法没有预先标注好的"正确答案"（即标签）。它的任务是探索数据内在的结构，将相似的数据点自动归入同一组（称为"簇"），同时让不同组之间的数据点尽可能不相似。

一个简单的比喻：想象你有一筐混合的水果，里面有苹果、橘子和香蕉。聚类算法的任务就是在没有人告诉你类别名称的情况下，自动把形状、颜色、大小相似的果子分别堆成一堆。

K-Means 算法的工作原理 ​

K-Means 是最常用、最直观的聚类算法之一。"K"代表我们想要将数据划分成的簇的数量。其工作原理可以概括为四个步骤：

  1. 初始化：随机选择 K 个数据点作为初始的"簇中心"（质心）。
  2. 分配：计算每个数据点到各个质心的距离（通常使用欧氏距离），然后将每个点分配到离它最近的质心所在的簇。
  3. 更新：重新计算每个簇的质心（即该簇所有点的平均值）。
  4. 迭代：重复步骤 2 和 3，直到质心的位置不再发生显著变化，或者达到预设的迭代次数。

下面的流程图清晰地展示了这个过程：

算法核心要点：

  • 距离度量：通常使用欧氏距离来衡量数据点之间的相似度，距离越近，相似度越高。
  • 质心：代表一个簇的"平均点"或中心点。
  • 目标：最小化每个簇内数据点到其质心的距离平方和（称为"簇内平方和"或 Inertia）。

数据缺失值处理 ​

# =========================================================================
# 3. 缺失值补充 (精准修复版：废弃 inplace=True)
# =========================================================================

# 3.1 PlayTimes 和 ASD 缺失直接显式填 0
df['PlayTimes'] = df['PlayTimes'].fillna(0)
df['ASD'] = df['ASD'].fillna(0)

# 3.2 根据 Target 字段的取值，动态填充 DD 字段的缺失值
if 'DD' in df.columns and 'Target' in df.columns:
    # 统一清洗一下 Target 字段，防止大小写或空格导致匹配不上
    df['Target'] = df['Target'].astype(str).str.strip()

    # 条件 1：当 DD 为空且 Target 为 Medium/medium 时，填入 Medium
    df.loc[df['DD'].isnull() & df['Target'].str.lower().isin(['medium']), 'DD'] = 'Medium'

    # 条件 2：当 DD 为空且 Target 为 Low/low 时，填入 Easy
    df.loc[df['DD'].isnull() & df['Target'].str.lower().isin(['low']), 'DD'] = 'Easy'

    # 条件 3：当 DD 为空且 Target 为 High/high 时，填入 Hard
    df.loc[df['DD'].isnull() & df['Target'].str.lower().isin(['high']), 'DD'] = 'Hard'

    # 兜底：如果 Target 本身也缺失导致 DD 没填上，默认给个中间态 Medium
    df['DD'] = df['DD'].fillna('Medium')

print("[检查] 执行完特定业务填充后，核心列的剩余空值数（必须全部为0）：")
print(df[['PlayTimes', 'ASD', 'DD']].isnull().sum())

结构换转 ​

py
# =========================================================================
# 4. 结构转化（特征编码）
# =========================================================================

# 为了防止数据中带有前后空格，先对字符型列进行清洗
for col in ['Sex', 'GameType', 'DD', 'Loc']:
    if col in df.columns and df[col].dtype == 'object':
        df[col] = df[col].astype(str).str.strip()

# 4.1 Sex 结构转化：Male -> 1, Female -> 0
sex_map = {'Male': 1, 'Female': 0, 'male': 1, 'female': 0}
df['Sex'] = df['Sex'].map(sex_map)

# 4.2 GameType 结构转化：Action -> 1, RPG -> 2, ...
gametype_map = {
    'Action': 1, 'action': 1,
    'RPG': 2, 'rpg': 2,
    'Simulation': 3, 'simulation': 3,
    'Sports': 4, 'sports': 4,
    'Strategy': 5, 'strategy': 5
}
df['GameType'] = df['GameType'].map(gametype_map)

# 4.3 DD 结构转化：easy -> 1, medium -> 2, hard -> 3
dd_map = {
    'easy': 1, 'Easy': 1,
    'medium': 2, 'Medium': 2,
    'hard': 3, 'Hard': 3
}
df['DD'] = df['DD'].map(dd_map)

# 4.4 Loc 结构转化：Asia -> 1, Eurpo -> 2, USA -> 3, Other -> 4
loc_map = {
    'Asia': 1, 'asia': 1,
    'Eurpo': 2, 'Europe': 2, 'europe': 2,
    'USA': 3, 'usa': 3,
    'Other': 4, 'other': 4
}
df['Loc'] = df['Loc'].map(loc_map)

# 4.5 安全检查与兜底填充（同步去除了 inplace 隐患）
df['Sex'] = df['Sex'].fillna(0)
df['GameType'] = df['GameType'].fillna(1)
df['DD'] = df['DD'].fillna(2)
df['Loc'] = df['Loc'].fillna(4)

print("\n[检查] 结构转化后各列的空值情况（必须全部为0）：")
print(df[['Sex', 'GameType', 'DD', 'Loc']].isnull().sum())

第 3 步:确定最佳簇数（K值） ​

K-Means 需要我们预先指定 K 值。如何选择一个合理的 K？我们使用两种经典方法：

1. 肘部法则 绘制不同 K 值对应的簇内平方和（Inertia）。Inertia 会随着 K 增大而减小，我们寻找曲线拐点（像手肘一样），其后的 K 值带来的收益（Inertia下降）变小。

2. 轮廓系数法 轮廓系数衡量一个数据点与自身簇的相似度（内聚度）和与其他簇的分离度。其值在 -1 到 1 之间，越高越好，表示聚类效果越佳。

py
#=========================================================================
#6. 确定最佳聚类数 K（手肘法与轮廓系数双重验证）
#=========================================================================
print("\n正在安全计算最优聚类数，请稍候...")

inertia = []
silhouette_scores = []
# ⚠️ 注意：由于 3 万条数据计算轮廓系数（Silhouette Score）需要消耗巨大计算量（O(N^2)）
# 如果在 Mac 上运行较慢，我们这里对样本进行 5000 条的快速分层抽样来做图评估，提升运行速度
sample_size = min(5000, len(X_scaled))
X_scaled_sample = X_scaled[:sample_size]

K_range = range(2, 7)

for k in K_range:
    kmeans = KMeans(n_clusters=k, init='k-means++', random_state=42, n_init=10)
    kmeans.fit(X_scaled)  # 训练依然使用全量数据
    inertia.append(kmeans.inertia_)

    # 评估使用抽样，防止 Mac 死机卡住
    score = silhouette_score(X_scaled_sample, kmeans.labels_[:sample_size])
    silhouette_scores.append(score)

# 可视化：手肘图与轮廓系数图
fig, ax1 = plt.subplots(figsize=(10, 5))

color = '#1f77b4'
ax1.set_xlabel('聚类数 K', fontsize=12)
ax1.set_ylabel('簇内平方和 (Inertia)', color=color, fontsize=12)
ax1.plot(K_range, inertia, 'o-', color=color, linewidth=2, label='Inertia')
ax1.tick_params(axis='y', labelcolor=color)
ax1.grid(True, linestyle='--', alpha=0.5)

ax2 = ax1.twinx()
color = '#2ca02c'
ax2.set_ylabel('轮廓系数 (Silhouette Score)', color=color, fontsize=12)
ax2.plot(K_range, silhouette_scores, 's--', color=color, linewidth=2, label='轮廓系数')
ax2.tick_params(axis='y', labelcolor=color)

plt.title('最优聚类数 K 评估图', fontsize=14, pad=15)
fig.tight_layout()
plt.show()

第 6 步：模型评估与应用建议 ​

评估：除了轮廓系数，可以查看簇内样本分布是否均衡，以及结合业务逻辑判断分群是否合理。

应用建议：

  • 精准营销：向"理想VIP型"（簇3）推送高端新品和独家活动；向"追求价值型"（簇2）发送折扣券和团购信息。
  • 产品开发：针对"年轻潜力型"（簇4）设计时尚、社交属性强的产品。
  • 客户服务：为"高收入谨慎型"（簇0）提供详细的产品数据和安全保障，打消其顾虑。
  • 资源分配：将更多客服和营销资源倾斜到高价值客户群。

总结与扩展 ​

通过这个实战案例，你已完整地体验了使用 K-Means 算法进行客户分群的流程：数据准备 -> 探索分析 -> 预处理 -> 确定K值 -> 训练模型 -> 分析结果。

关键要点回顾：

  • 聚类是无监督学习，用于发现数据内在分组。
  • 特征标准化是使用基于距离的聚类算法前的关键步骤。
  • 肘部法则和轮廓系数是确定最佳簇数的实用工具。
  • 聚类结果的解读必须结合业务知识，才能产生真正的价值。

TIP

扩展探索] 开发者可根据业务需求，动态过滤 Target 标签（Low/Medium/High）以锁定特定风险或价值群体。同时，支持自定义调整 features_to_cluster 特征矩阵，实现多维度的微观用户画像深度下钻与探索。

py
# 导入必要的库
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import silhouette_score
from sklearn.decomposition import PCA
import warnings
warnings.filterwarnings('ignore') # 忽略非关键警告

# 设置可视化风格
sns.set_style("whitegrid")
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'PingFang SC', 'Heiti TC', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

# =========================================================================
# 1 & 2. 加载数据并精准筛选 Target == '' 的用户
# =========================================================================
df_raw = pd.read_excel('DS_train.xlsx')
print("原始数据形状:", df_raw.shape)

# 核心修改：统一清洗大小写和空格后，只保留 用户
df_raw['Target'] = df_raw['Target'].astype(str).str.strip().str.lower()
# Low Medium  High
tar='high'
df = df_raw[df_raw['Target'] == tar].copy().reset_index(drop=True)

print(f"👉 【已筛选】仅包含 Target 为 {tar} 的用户，分析样本数: {df.shape[0]} 行")

# =========================================================================
# 3. 缺失值补充 (针对 Low 用户群)
# =========================================================================
df['PlayTimes'] = df['PlayTimes'].fillna(0)
df['ASD'] = df['ASD'].fillna(0)

# 因为已经全是  用户了，根据你之前的业务逻辑：Target 填 Low 则 DD 补 Easy
# Easy Medium Hard
# df['DD'] = df['DD'].fillna('Medium')

print("\n[检查] 核心列剩余空值数：\n", df[['PlayTimes', 'ASD', 'DD']].isnull().sum())

# =========================================================================
# 4. 结构转化（特征编码）
# =========================================================================
for col in ['Sex', 'GameType', 'DD', 'Loc']:
    if col in df.columns and df[col].dtype == 'object':
        df[col] = df[col].astype(str).str.strip()

df['Sex'] = df['Sex'].map({'Male': 1, 'Female': 0, 'male': 1, 'female': 0}).fillna(0)
df['GameType'] = df['GameType'].map({'Action': 1, 'action': 1, 'RPG': 2, 'rpg': 2, 'Simulation': 3, 'simulation': 3, 'Sports': 4, 'sports': 4, 'Strategy': 5, 'strategy': 5}).fillna(1)
df['DD'] = df['DD'].map({'easy': 1, 'Easy': 1, 'medium': 2, 'Medium': 2, 'hard': 3, 'Hard': 3}).fillna(1) # 默认为 Low 对应的 Easy(1)
df['Loc'] = df['Loc'].map({'Asia': 1, 'asia': 1, 'Eurpo': 2, 'Europe': 2, 'europe': 2, 'USA': 3, 'usa': 3, 'Other': 4, 'other': 4}).fillna(4)

# =========================================================================
# 5. 特征工程补充（过滤与标准化）
# =========================================================================
# 剔除 Sex 和 GameType 和 DD，保留以下核心行为/环境特征进行微观特征挖掘
features_to_cluster = ['Age','GameType', 'Loc', 'PlayTimes', 'InGame', 'SPK', 'ASD', 'Level', 'ACHV']

# 终极安全填充
for col in features_to_cluster:
    df[col] = df[col].fillna(df[col].median())

X = df[features_to_cluster].copy()
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# =========================================================================
# 6. 确定  用户群的最佳聚类数 K（自动运行评估，供你参考拐点）
# =========================================================================
print(f"\n正在评估 {tar} 用户群的最优聚类数...")
inertia = []
silhouette_scores = []
sample_size = min(4000, len(X_scaled))
X_scaled_sample = X_scaled[:sample_size]

K_range = range(2, 7)
for k in K_range:
    kmeans = KMeans(n_clusters=k, init='k-means++', random_state=42, n_init=10)
    kmeans.fit(X_scaled)
    inertia.append(kmeans.inertia_)
    silhouette_scores.append(silhouette_score(X_scaled_sample, kmeans.labels_[:sample_size]))

# 绘制评估图
fig, ax1 = plt.subplots(figsize=(10, 4))
ax1.set_xlabel('聚类数 K')
ax1.set_ylabel('Inertia', color='#1f77b4')
ax1.plot(K_range, inertia, 'o-', color='#1f77b4')
ax2 = ax1.twinx()
ax2.set_ylabel('Silhouette Score', color='#2ca02c')
ax2.plot(K_range, silhouette_scores, 's--', color='#2ca02c')
plt.title(f'【{tar} 用户群专用】最优聚类数 K 评估图')
fig.tight_layout()
plt.show()

# =========================================================================
# 7. 最终  用户群模型训练与详尽画像输出
# =========================================================================
# 💡 针对  用户的微观分类，分成 3 类或 4 类通常最具有细分运营意义。这里先默认设为 3 类。
low_k = 3

kmeans_low = KMeans(n_clusters=low_k, init='k-means++', random_state=42, n_init=10)
df['Low_Cluster'] = kmeans_low.fit_predict(X_scaled)

print(f"\n[聚类完成] {tar}  用户群内部的分群数量分布：")
print(df['Low_Cluster'].value_counts())

cluster_profile = df.groupby('Low_Cluster')[features_to_cluster].mean()
pd.set_option('display.max_columns', None)
pd.set_option('display.width', 1000)

print(f"\n🔥 【{tar}  用户群内部细分】核心画像特征均值：")
print(cluster_profile.round(2))

# =========================================================================
# 8. 绘制  用户群聚类散点图
# =========================================================================
print(f"\n正在生成 {tar}  用户专用聚类可视化图表...")
pca = PCA(n_components=2, random_state=42)
X_pca = pca.fit_transform(X_scaled)
df['PCA1'] = X_pca[:, 0]
df['PCA2'] = X_pca[:, 1]

plt.figure(figsize=(10, 7), dpi=100)

    x='PCA1', y='PCA2', hue='Low_Cluster', data=df,
    palette=custom_colors, alpha=0.6, edgecolor='none', s=15

plt.title('Target="Low" 风险用户微观细分空间分布图', fontsize=15, fontweight='bold', pad=15)
plt.xlabel('主成分 1 - 综合行为轴', fontsize=12)
plt.ylabel('主成分 2 - 活跃偏离轴', fontsize=12)
